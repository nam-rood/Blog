避坑

首先我买了账号，在邮箱中注册之后，网页版的claude是可以使用的，但是claude code是无法使用的！！查了一晚上，也不知道为什么，就显示我账号无法登陆，最后发现是由于之前使用过了cc-switch，环境变量被他修改了，直接使用claudecode，就会使用他的环境变量。导致出错。只需要去清理.claude/setting.json文件即可，修改为你的vpn的出口。我的是这样的。

```
{
  "env": {
    "HTTP_PROXY": "http://127.0.0.1:7890",
    "HTTPS_PROXY": "http://127.0.0.1:7890"
  },
  "model": "sonnet"
}
```

还有就是，用账号的方式使用claudecode，不是以API的方式！！！因为我之前使用claudecode连接过智谱的glm-4.6，所以以为claude也是使用API的方式，并且还去创建key，去配置。最后发现显示API额度不足。后来才发现，claude code直接登陆账号就可以使用了。

vpn问题。我最先排查的就是vpn，去查看了一下，发现我的vpn共享10000+，就想着去换一个VPN，因为我现在这个真的超级便宜。。15r/200G，不限时。但最后误打误撞，没有去换vpn，现在也能正常使用。可能后期会被封号吧。
